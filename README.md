# sw-cli

Um cliente para a **Star Wars API** (https://swapi.co/)

## Como se aventurar

Clonar e executar projeto

```
$ git clone https://github.com/nettooe/sw-cli
$ cd sw-cli
$ ./mvnw clean install spring-boot:run
```

Em seguida, acesse:

http://localhost:8080/swagger-ui.html
